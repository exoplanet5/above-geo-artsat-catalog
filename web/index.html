Catalog Entries

Format column — Classical: elements (inclination, RAAN, eccentricity, mean motion, perigee, apogee) parsed from a standard 80-column TLE. Extended: Bill Gray’s extended-format TLE used when the object is too deep-space for the standard column widths; elements are not parseable from the line, so these rows are labelled as the deep-space zone and are excluded from the element-based plots.

Source: Bill Gray’s high-altitude artificial-satellite TLE list, part of Project Pluto — projectpluto.com/tle_info.htm . The tle_list.txt manifest and the referenced .tle files drive every row in this catalog.
